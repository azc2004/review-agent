# Review AI - 스마트 상품평 생성기 📸

사용자가 구매한 상품의 사진을 업로드하면, OpenAI Vision LLM(Multimodal AI)이 이미지를 정밀히 분석하여 실제 고객이 작성한 것과 같은 자연스럽고 입체적인 쇼핑몰 리뷰 초안을 생성해 주는 Streamlit 기반 웹 애플리케이션입니다.

## 🌟 핵심 가치 (Value Proposition)
- **리뷰 작성 피로도 감소:** 텍스트를 한 줄 한 줄 고민해서 작성할 필요 없이, 사진만 올리면 맞춤 초안을 3초 만에 제공합니다.
- **가짜 리뷰 논란 방지 (Human-in-the-loop):** AI가 완전히 대필하여 자동 등록하는 대신, 사용자에게 문장 초안을 제공하고 사용자가 직접 수정 및 편집을 거쳐 등록하도록 하여 신뢰성을 확보합니다.
- **맥락에 맞는 고품질 리뷰:** 단순 텍스트 프롬프트 기반 생성을 넘어, 실제 업로드된 이미지 속 상품의 색감, 질감, 디자인, 상태 등을 다각도로 분석하여 구체적인 리뷰를 작성합니다.

---

## 🛠️ 주요 기능 (Features)
1. **📸 이미지 업로드 및 미리보기**
   - JPG, PNG, JPEG, WEBP 형식의 상품 이미지를 손쉽게 업로드하고 실시간으로 미리볼 수 있습니다.
2. **🤖 OpenAI Vision LLM 기반 분석**
   - `gpt-4o` 및 `gpt-4o-mini` 모델 중 원하는 엔진을 선택하여 이미지를 디테일하게 스캔합니다.
3. **✍️ 다양한 리뷰 스타일 지원**
   - **정중한 존댓말**: 예의 바른 격식체 적용
   - **친근한 반말**: 친근하고 편안한 SNS/블로그 스타일
   - **솔직담백한 한달 후기**: 장단점을 솔직하게 담은 1개월 사용기 형태
   - **블로그 체험단 스타일**: 이모티콘을 활용한 풍부하고 활기찬 묘사
4. **🎯 선택적 핵심 키워드 강조**
   - 디자인 및 색상, 재질 및 촉감, 가성비/가격, 배송 상태 및 포장, 조립/사용 편의성, 사이즈 및 핏 등 원하는 키워드를 다중 선택하여 관련 내용 위주로 본문을 작성할 수 있습니다.
5. **⭐ 별점 기반 감정선 반영**
   - 별점(5점 ~ 1점)에 맞추어 칭찬 가득한 긍정 후기부터 장단점 절반씩인 중립 후기, 그리고 품질 문제나 실망스러운 팩트에 근거한 솔직한 부정 후기까지 톤앤매너를 영리하게 조절합니다.
6. **📝 실시간 편집 및 복사**
   - AI가 추출한 핵심 `#태그` 키워드를 한눈에 보여주고, 편집기(`st.text_area`)에서 즉시 텍스트를 원하는 대로 수정할 수 있습니다.

## 🚀 빠른 시작 가이드 (Quick Start)

### 1. 가상환경 설정 및 활성화
프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 가상환경을 구축하고 활성화합니다.

```bash
# 가상환경 생성
python -m venv venv

# 가상환경 활성화 (macOS/Linux)
source venv/bin/activate

# 가상환경 활성화 (Windows CMD)
# venv\Scripts\activate.bat

# 가상환경 활성화 (Windows PowerShell)
# .\venv\Scripts\Activate.ps1
```

### 2. 의존성 패키지 설치
`requirements.txt`에 명시된 필수 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

### 3. 환경 변수 설정
프로젝트 루트 디렉토리에 `.env` 파일을 생성하고, 발급받은 OpenAI API 키를 입력합니다.
프로젝트 내의 `.env.example` 파일을 복사해 사용할 수 있습니다.

```bash
cp .env.example .env
```

**.env 파일 예시:**
```env
OPENAI_API_KEY=your_openai_api_key_here
```

### 4. 애플리케이션 실행
설정이 완료되었으면 Streamlit 애플리케이션을 가동합니다.
```bash
streamlit run app.py
```
실행이 완료되면 자동으로 브라우저에 `http://localhost:8501` (혹은 사용 중인 포트) 페이지가 열립니다.

## 🖥️ UI 사용 가이드 (How to Use)
1. **API 키 구성:** 사이드바 상단에 OpenAI API 키를 입력하거나, `.env` 파일에 사전에 키를 입력해 두면 자동으로 감지하여 초록색 성공 배지가 나타납니다.
2. **리뷰 스타일 선택:** 사이드바에서 원하시는 **AI 모델(추천: gpt-4o-mini)**, **어조**, **강조 키워드**, 그리고 **별점**을 알맞게 설정합니다.
3. **이미지 업로드:** 메인 좌측 영역의 '상품 사진 업로드' 위젯을 통해 사진을 드래그 앤 드롭 하거나 탐색기로 업로드합니다.
4. **리뷰 생성 및 편집:** 메인 우측 영역의 `AI 리뷰 초안 만들기` 버튼을 클릭하면 AI가 실시간으로 분석하여 키워드 태그와 에디터에 작성된 초안을 출력합니다.
5. **완성 및 복사:** 에디터 내부의 글을 가볍게 다듬어 쇼핑몰 리뷰란에 붙여넣어 사용하면 끝납니다!
