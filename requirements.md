# AI 기반 상품평 리뷰 생성 서비스 (프로토타입) 작업지시서

## 1. 서비스 개요
- **목적:** 사용자가 구매한 상품의 사진을 업로드하면, Vision LLM(Multimodal AI)이 이미지를 분석하여 상품평 작성을 도와주는 초안(템플릿)을 제공하여 리뷰 작성의 허들을 낮춥니다.
- **타겟 사용자:** 상품평 작성을 번거로워하지만 쇼핑몰 포인트/리워드는 받고 싶어하는 소비자.
- **주요 가치:** 리뷰 작성 시간 단축, 구매 전환율을 높이는 고품질 사진 리뷰 확보.

## 2. 최신 트렌드 조사 요약
현재 AI 이커머스 리뷰 생성/분석 서비스의 주요 트렌드는 다음과 같습니다.
1. **작성 보조(Drafting) 및 투명성:** AI가 완벽한 리뷰를 대필하여 '가짜 리뷰' 논란을 일으키는 대신, 사진에서 키워드(예: 핏, 소재, 색상 등)를 추출해 **문장 초안(템플릿)을 제공하고 사용자가 직접 수정**하도록 유도하는 'Human-in-the-loop' 방식이 주류입니다.
2. **멀티모달(Multimodal) AI 적극 활용:** 텍스트 프롬프트뿐 아니라, 사용자가 업로드한 **사진 자체**에서 상품의 상태, 색감, 질감, 특징 등을 분석하여 맥락에 맞는 리뷰 내용을 생성합니다.
3. **장단점(Pros/Cons) 및 요약 위주:** 긴 줄글보다는 요약된 불릿 포인트(Bullet Points) 및 장단점 위주의 가독성 높은 리뷰 포맷을 선호합니다.

## 3. 핵심 기능 요구사항
1. **사진 업로드 기능:** 사용자가 상품 사진(jpg, png, webp 등)을 업로드하고 미리볼 수 있어야 합니다.
2. **이미지 분석 (Vision LLM 연동):** 
   - 업로드된 이미지를 Multimodal 모델(OpenAI GPT-4o 등)로 전달합니다.
   - **프롬프트 전략:** "이 사진의 상품 종류를 식별하고, 보이는 특징(색상, 재질, 디자인, 상태 등)을 바탕으로 실제 고객이 작성한 것 같은 쇼핑몰 리뷰 초안을 작성해줘. 장점과 단점을 포함한 키워드 태그들과, 3~4문장 분량의 텍스트 초안을 마크다운으로 제공해줘."
3. **리뷰 초안 템플릿 출력 및 편집:** AI가 분석한 결과를 텍스트 에디터(`st.text_area` 등)에 렌더링하여, 사용자가 입맛에 맞게 내용을 바로 수정할 수 있어야 합니다.
4. **리뷰 등록 시뮬레이션:** 수정이 끝난 후 '리뷰 등록하기' 버튼을 누르면 성공 메시지가 뜨도록 구현합니다.

## 4. UI/UX 요구사항 (Streamlit 기반)
- **화면 구성:**
  - **헤더:** 서비스 타이틀 및 안내 ("📸 사진만 올리면 AI가 리뷰를 써드려요!")
  - **입력부:** 사진 업로드 위젯 (`st.file_uploader`) 및 업로드 이미지 화면 표시 (`st.image`)
  - **제어부:** 'AI 리뷰 자동 생성' 버튼
  - **출력부:** 분석 중 로딩 스피너 (`st.spinner`) -> 추출된 키워드 표시 -> 편집 가능한 텍스트 에디터에 리뷰 초안 제공
  - **완료부:** 최종 '리뷰 등록' 버튼 및 Success 메시지
- **디자인:** Streamlit의 기본 기능을 활용하여 모바일/웹 모두에서 깔끔하게 보이도록 레이아웃(`st.columns` 등) 구성.

## 5. 기술 스택 (제안)
- **Language:** Python 3.9+
- **Frontend Framework:** Streamlit
- **AI API:** `openai` (GPT-4o)
- **Image Processing:** `Pillow` (PIL)
